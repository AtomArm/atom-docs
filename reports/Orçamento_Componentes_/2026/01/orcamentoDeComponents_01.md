### Justificativa dos Componentes

- **Componentes Mecânicos**: Os guias lineares e rolamentos são essenciais para garantir o movimento suave e preciso das peças móveis do ATOM Chess. O fuso axial é necessário para a movimentação vertical, enquanto os parafusos, arruelas e porcas são fundamentais para a montagem e fixação dos componentes mecânicos.

- **Componentes Eletrônicos**: A fonte de alimentação é crucial para fornecer energia estável aos componentes eletrônicos. Os motores de passo e servo são necessários para o controle dos movimentos do robô, enquanto a placa de controle (Banana Pi e Raspberry Pi) é essencial para a execução do software de controle do ATOM Chess.

- **Componentes de Sensores e Controle**: Os sensores de corrente e os conversores de nível lógico são importantes para monitorar o consumo de energia e garantir a compatibilidade entre os componentes eletrônicos. O módulo USB-TTL é necessário para a programação e depuração dos microcontroladores, enquanto o driver PCA9685 é essencial para controlar os servos de forma eficiente.
